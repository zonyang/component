drag：拖拽组件
  • onmousedown与onclick区别：onclick是获取锚点 可以是鼠标、tab键；onmousedown只能是鼠标
  • offset系列 
    1. offsetParent 最近的已经定位的祖先元素 无定位祖先则是body
    2. offsetWidth = width+ padding +border
    3. offsetHeight 同上
    4. offsetTop 相对于最近的已经定位的祖先元素的top值差
    5. offsetLeft 同上
  • 事件三要素
    1. 事件源 e.target (标准) || window.event.srcElement (IE)
    2. 事件处理函数 function
    3. 事件 e = e (标准) || window.event 包含多种属性
  • e.clientX：返回当事件被触发时鼠标指针向对于浏览器页面（或客户区）的水平坐标。
  • null与undefined：定义一个变量未赋初始值时为undefined；获取一个不存在的元素为null；
  • 被拖拽组件的位置设置
    1. 移动距离 = clientY终值 - clientY初值；实际top值 = 移动距离 + 初始top值（包含框有定位则相对于包含框 无则相对于body）；  任意位置可以拖拽的框必有初始值，用以下方法更好
    2. 鼠标距离框顶部top = clientY初值 - 框offsetTop
       实际top值 = clientY终值 - 鼠标距离框顶部top
